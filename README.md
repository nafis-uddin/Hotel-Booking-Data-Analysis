# Hotel Booking Data Analysis

## Project Overview
Analyzed hotel booking data to uncover customer behavior, booking trends, cancellations, and revenue opportunities using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objectives
- Analyze booking patterns
- Identify cancellation factors
- Explore customer segments
- Generate business insights

## Key Business Insights
- Peak booking seasons identified
- Cancellation trends analyzed
- Customer distribution by hotel type
- Average stay duration evaluated
